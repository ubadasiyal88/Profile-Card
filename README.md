# 👤 Profile Card UI Design

A clean and modern **Profile Card UI** built using only **HTML and CSS**.
This project showcases a simple yet elegant user profile layout with a responsive design and smooth visual styling.

---

## 📌 Features

* 🎨 Modern and attractive UI design
* 👤 Profile image with styled layout
* 🔗 Social media icons section
* 🔘 Interactive buttons (Subscribe & Message)
* 📊 User stats (likes, comments, shares)
* 📱 Responsive and centered layout

---

## 🚀 Technologies Used

* HTML5
* CSS3 (Flexbox & Styling)

---

## 🎯 Purpose of the Project

This project was created to practice and improve:

* CSS layout techniques
* UI design skills
* Component-based design
* Responsive styling
